# Using OCCA

## Preface

The folders in this directory contain a series of progressive examples demonstrating the fundamentals of the OCCA framework through to more complex use cases. Each example is meant to be self-contained, only requiring an OCCA installation and the source code in the corresponding folder.

Working through all of the examples in-order offers the most comprehensive coverage; however, several learning paths are outlined below in an attempt to expedite learning for some common categories of developers.
